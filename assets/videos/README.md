# Welcome Video

Place your welcome video file here with the name `welcome.mp4`.

## Video Requirements:
- Format: MP4
- Resolution: 1080x1920 (9:16 aspect ratio for mobile)
- Duration: 10-15 seconds recommended
- File size: Keep under 10MB for optimal loading
- Content: Should showcase your app's key features or branding

## Supported Formats:
- MP4 (recommended)
- MOV
- AVI

The video will play automatically when the onboarding screen loads and will show the last frame when it ends (no looping).
